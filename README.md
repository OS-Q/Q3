#### HASH

包括BLAKE2算法和SHA256

```

BLAKE2的定位是目前安全系数最高的哈希函数。BLAKE2是基于BLAKE实现的，BLAKE是2008年被提交至SHA-3竞赛的一种哈希函数。

BLAKE2b是BLAKE的64位版本，它可以生成最高512位的任意长度哈希。BLAKE2s是BLAKE的32位版本，它可以生成最高256位的任意长度哈希。

```
